Regions: Spatial Region Filtering

Summary

In astronomical data analysis, it often is useful to select spatial regions of an image for further processing. The regions library provides support for filtering FITS images and binary tables using simple geometric shapes, including boolean operations between shapes. The regcnts program uses this library to support background subtracted counts in regions.
